##  Project Overview

This project was completed as part of the Digital Literacy course. As a Student Digital Ambassador, the objective was to develop awareness about digital tools, online safety, professional communication, and cybercrime prevention.

The project consists of five tasks that focus on practical skills such as designing an infographic, building a digital portfolio, practicing coding, understanding collaboration tools, and promoting cybersecurity awareness. All tasks are organized in this repository along with supporting files and a detailed project report.



---

##  Task Summary

### 🔹 Task 1: Digital Literacy Infographic
A one-page infographic was created using Canva to explain digital literacy concepts, useful tools, safe internet practices, professional online presence, and email etiquette in a visually engaging format.

---

### 🔹 Task 2: Digital Portfolio
Profiles were created on GitHub, LinkedIn, and Kaggle to establish a professional digital presence. These platforms help in showcasing skills, networking, and tracking learning progress.

---

### 🔹 Task 3: Coding & Collaboration
A beginner-level coding challenge (“Solve Me First”) was completed on HackerRank to improve programming skills. Additionally, a Google Form quiz was created to understand collaboration and data collection tools.

---

### 🔹 Task 4: Email Etiquette
Two professional emails were drafted:
- Assignment extension request  
- Internship application  

A social media checklist was also created to promote responsible online behavior.

---

### 🔹 Task 5: Cybercrime Awareness
A case study on UPI fraud was developed, along with a prevention checklist to educate students about safe online practices and cybercrime reporting methods.
